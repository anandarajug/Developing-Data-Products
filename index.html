About the App

This App has been prepared as part of Developing Data Products Course Project.

App is intended to demonstrate text animation entered by the user and a character count will be displayed at the bottom. This App also uses javascript.

How to Use the App

User has to type-in the desired text in the given textbox and then click on Animate button in the left pane to see the animation in action. Result will be seen in right pane.

Text annotation will printed mentioning what the user has typed in the Text Box

User has to type some text that he wish to animate.
